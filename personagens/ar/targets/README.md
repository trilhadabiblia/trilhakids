# Targets MindAR

Esta pasta contém os arquivos `.mind` compilados para rastreamento de imagem.

## Como gerar o sansao.mind

1. Acesse: https://hiukim.github.io/mind-ar-js-doc/tools/compile
2. Clique em **Upload Images** e envie `personagens/img/sansao.png`
3. Clique em **Start**
4. Baixe o arquivo e renomeie para `sansao.mind`
5. Coloque-o nesta pasta

## Para novos personagens

Repita o processo com a imagem do personagem desejado e salve como `[nome].mind`.
