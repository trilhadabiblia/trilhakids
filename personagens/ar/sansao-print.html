Passo a passo

Como usar este cartão AR

  1. Imprima esta página (botão abaixo)
  2. Acesse hiukim.github.io/mind-ar-js-doc/tools/compile
  3. Faça Print Screen / screenshot desta página e envie ao compilador
  4. Baixe o .mind, renomeie para sansao.mind
  5. Suba em personagens/ar/targets/sansao.mind
  6. Abra ar/sansao.html e aponte para o cartão impresso
